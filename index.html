O que é Programação Linear?

Programação Linear (PL) é uma técnica de otimização que busca maximizar ou minimizar uma função objetivo linear sujeita a um conjunto de restrições lineares. É amplamente aplicada em logística, finanças, engenharia de produção e ciência de dados.

Forma Geral de um PL
Maximizar (ou Minimizar): c₁x₁ + c₂x₂ + ... + cₙxₙ

Sujeito a:
  a₁₁x₁ + a₁₂x₂ + ... + a₁ₙxₙ ≤ b₁
  a₂₁x₁ + a₂₂x₂ + ... + a₂ₙxₙ ≤ b₂
  ...
  xⱼ ≥ 0, para todo j = 1, ..., n
